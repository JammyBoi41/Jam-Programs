Projects

Jam's Powerlifting App - A personal project that combines a passion for fitness and software development, this application's back-end was written in Python and front-end development was done with Qt(PySide6). It incorporates the Python package sqlite3 for storing user information. It makes use of a public database to provide the user with a pre-loaded list of food options to log for convenience, while also allowing them to log custom meals as well. Additionally, it supports Personal Record tracking for the 3 powerlifts, as well as nutritional display for tracking calories and protein intake, all in a custom, simplified interface.

Flight Delay Calculator - An average of one out of five flights get delayed every day. Due to these flight setbacks, our team developed a flight delay calculator using an existing database of flight times in order to calculate average flight delay time between 2 cities as well as their frequency of delay. In order to implement this in an efficient manner, we implemented our own Hash Table and Graph structures to allow for near-instant lookup, allowing for insight into both how likely and how long your flight is likely to be delayed for.
